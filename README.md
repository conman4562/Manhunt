## Manhunt datapack for 1.21.6 - 1.21.11\
Mimicking dreams manhunts as closely as possible

To install:\
Download the zip, extract it, and put it in your world's datapack folder

To join teams:\
```/team join <playername> hunters```\
```/team join <playername> runners```

you can modify the start_game.mcfunction file to automatically join teams on start\
you can also modify that file to change any start game states (i.e. time, gamerules, etc.)

To start the game:\
```/function start_game```

Right click the compass to update it\
There is no win detection or headstart functionality, as it is intended to mimic dream style manhunts\
There is game over detection that will automatically stop the function upon game end (all runners dead or dragon dead)
